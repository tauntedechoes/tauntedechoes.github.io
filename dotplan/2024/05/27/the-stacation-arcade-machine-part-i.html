.plan - The Stacation Arcade Machine, Part I

2024-05-27

Had some time while taking a little home-based vacation, so I decided to start in on an acade machine.

Notes until I find time to actually expand this into a written thing.

  • Always wanted my own arcade machine
  • Fiddled around with Mame a few times over the years.
  • Built an actual one with a bunch of spare parts at zu.
  • Played on a friend's set up a couple of weeks ago.
  • Said screw it, time to start building my own.
  • Found a romset for the latest mame, downloaded it, and figured out how to make it run to test out games that I wanted to play.
  • Figured out how to boot my steamdeck into Batocera, how to update bios-es and roms, and how to update the system to a dev version to get the latest version of mame.
  • Now I have a bare-bones arcade machine that runs games.
  • Next step... arcade controller!
